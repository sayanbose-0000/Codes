1. Creating react app with CRA (Create React App)-

- `npx create-react-app my-app`
- `cd my-app`
- `npm start`



2. Using vite-

- `npm create vite@latest`
- give the project name. eg `my-app`, select `react` and then select `javascript + swc` (swc is speedy web compiler)
- `cd my-app`
- `npm install`
- `npm run dev`
- use `npm run dev` everytime we want to open it in chrome and `ctrl+click` on link to open in chrome
