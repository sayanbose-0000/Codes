Types of access modifiers
-------------------------

1)
public- class, package, subpackage, world 
Basically accessible everywhere


2)
no modifier- classes with in the same package 
NOT visible to classes in different package


3)
protected- class, package, subpackage
It can be accessed in a second packages's class as long as that particular class is a subclass(child class) of a class of the first package


4)private- only within the class it has been declared
